1. Introduction

This Privacy Policy describes how Qopy ("we," "our," or "us") collects, uses, and protects your personal information when you use our clipboard manager application (the "Application"). Qopy is an open-source clipboard manager designed for Windows, macOS, and Linux systems.

2. Information We Collect

2.1 Clipboard Data

  • What we collect: Qopy stores clipboard content that you copy, including text, images, files, and other data types supported by your operating system's clipboard functionality.
  • How it's stored: All clipboard data is stored locally on your device in a SQLite database.
  • Duration: Clipboard history is maintained according to your configured settings and may be cleared manually or automatically based on your preferences.

2.2 Application Usage Data

  • Local Configuration: Settings, preferences, and application configuration data
  • Performance Data: Local application performance metrics for optimization purposes
  • Error Logs: Local crash reports and error logs to improve application stability

2.3 System Information

  • Operating System: Basic OS information for compatibility and feature support
  • Hardware Information: Basic hardware specifications for performance optimization
  • Application Version: Current version information for update management
  • Technical Performance Metrics: System performance data, response times, and error rates (no content analysis)

2.4 Sensitive Data

  • Special Categories: We may process special categories of personal data if you copy sensitive information (health data, biometric data, etc.) to your clipboard
  • Legal Basis: Processing based on explicit consent or substantial public interest where applicable
  • Additional Safeguards: Enhanced security measures apply to any sensitive data processed

3. Legal Basis for Data Processing (GDPR Article 6)

3.1 Lawful Basis

  • Legitimate Interest (Article 6(1)(f)): Processing clipboard data for the core functionality of the clipboard manager service
  • Contract Performance (Article 6(1)(b)): Processing necessary to provide the clipboard management service you've requested
  • Consent (Article 6(1)(a)): Where explicitly requested for optional features like analytics or marketing communications

3.2 Balancing Test

We have conducted a legitimate interest assessment and determined that our processing is necessary, proportionate, and does not override your fundamental rights and freedoms.

4. How We Use Your Information

4.1 Primary Functions

  • Clipboard Management: To provide clipboard history, search, and management functionality
  • Data Storage: To maintain your clipboard history within the application
  • Application Operation: To ensure proper functioning of the application features
Important: We do not perform any analysis, statistics, or data mining on your clipboard content. No statistical data is generated from your copied content.

4.2 Application Improvement

  • Performance Optimization: To improve application performance and user experience through technical metrics only
  • Feature Development: To develop new features and enhance existing functionality based on user feedback, not content analysis
  • Bug Fixes: To identify and resolve technical issues through error logs and system diagnostics
Data Content Exclusion: We explicitly do not analyze, process, or generate statistics from the actual content of your clipboard items. All application improvements are based solely on technical performance data and user feedback.

5. Data Storage and Security

5.1 Cloud Infrastructure (Railway.app)

  • Hosting Provider: Qopy is hosted on Railway.app infrastructure
  • Data Location: Application data is processed and stored on Railway's servers in Amsterdam, Netherlands
  • Database: Clipboard data is stored in cloud-hosted databases managed through Railway's platform
  • Geographic Location: EU West region (Amsterdam, Netherlands) ensuring GDPR compliance and data residency within the European Union

5.2 Railway.app Infrastructure Security

  • SOC 2 Compliance: Railway.app maintains SOC 2 Type I certification
  • Shared Responsibility: Security follows a shared responsibility model between Qopy and Railway
  • Infrastructure Security: Railway provides enterprise-grade infrastructure security measures
  • Data Processing Agreement: Railway operates under a Data Processing Agreement (DPA) for GDPR compliance

5.3 Application Security Measures

  • Encryption in Transit: All data transmission is encrypted using TLS/SSL
  • Access Controls: Strict access controls limit who can access your clipboard data
  • Data Isolation: User data is isolated between different users and sessions
  • Regular Security Updates: Both application and infrastructure receive regular security updates

5.4 Data Retention Periods

  • Clipboard Content: Retained according to user-configured settings (default: 30 days)
  • User Account Data: Retained for the duration of account activity plus 12 months
  • System Logs: Retained for 90 days for security and operational purposes
  • Backup Data: May be retained up to 1 year in encrypted backup systems
  • Legal Hold: Data may be retained longer if subject to legal obligations

5.5 Automated Deletion

  • Scheduled Cleanup: Automated deletion of expired clipboard items
  • User-Triggered: Immediate deletion when users manually clear data
  • Account Deletion: Complete data removal within 30 days of account deletion request

5.6 Data Minimization Principles

  • Purpose Limitation: Data only processed for specified, legitimate purposes
  • Storage Limitation: Data retained no longer than necessary for the purposes
  • Accuracy: Measures to ensure data accuracy and keep it up to date

6. Data Breach Response

6.1 Incident Response Plan

7.1 No Third-Party Sharing

  • Right to Data Portability (Article 20): Receive your data in a structured, machine-readable format
  • Right to Object (Article 21): Object to processing based on legitimate interests
  • Rights Related to Automated Decision-Making (Article 22): Protection against solely automated decision-making

8.2 Exercising Your Rights
