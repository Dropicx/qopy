Dual Licensing Policy

Qopy is available under two licensing options to accommodate different use cases and requirements. This dual licensing approach allows us to support both open source development and commercial applications.

Open Source License (AGPL-3.0)

FREE

Perfect for: Personal projects, educational use, open source development, and internal company tools that are not distributed.

Key Features:

  • Completely free - No cost for use
  • Full source code access - View and modify the code
  • Community-driven - Contribute to the project
  • Internal use - Use within your organization

Requirements:

  • Source code disclosure - Must share source code of derivative works
  • AGPL-3.0 compliance - Combined works must use AGPL-3.0
  • Web service disclosure - Must provide source code for web services using this software

License Text: View AGPL-3.0 License

Commercial License

PAID

Perfect for: Proprietary software, SaaS applications, commercial products, and when you cannot comply with AGPL-3.0 requirements.

Key Benefits:

  • No copyleft restrictions - No source code disclosure requirements
  • Proprietary integration - Integrate into closed-source software
  • SaaS deployment - Offer as web service without source disclosure
  • Commercial support - Professional support and warranty included
  • Redistribution rights - Include in commercial products

Commercial Use Includes:

  • Production business operations
  • Proprietary application integration
  • SaaS and web services
  • Commercial product redistribution
  • Internal tools that are distributed
